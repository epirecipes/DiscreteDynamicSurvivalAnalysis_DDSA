# DiscreteDynamicSurvivalAnalysis

This repository contains examples of discrete dynamic survival analysis.

## Synthetic data case scenarios:

-[DDSA data-availability scenarios](https://github.com/epirecipes/DiscreteDynamicSurvivalAnalysis_DDSA/blob/main/Data_avail_GLCT_Boxcar_Geo/ddsa_data_avail.md): This notebook evaluates each likelihood component and what they contribute by fitting a simple SIR DDSA model under three data-availability scenarios.

## Real-world case scenarios:

-[H1N1 outbreak](https://github.com/epirecipes/DiscreteDynamicSurvivalAnalysis_DDSA/blob/main/H1N1_WSU/dsa_WSU_Turing.md): H1N1 outbreak at Washington State University (WSU) in 2009.
-[Ebola outbreak](https://github.com/epirecipes/DiscreteDynamicSurvivalAnalysis_DDSA/blob/main/Ebola_DRC_2018/DSA_turing_SIRmap_W2.md): The 2018–2020 Ebola virus disease (EVD) epidemic in the Democratic Republic of the Congo (DRC).
-[Measles outbreak](https://github.com/epirecipes/DiscreteDynamicSurvivalAnalysis_DDSA/blob/main/Measles_Hagelloch_1861/dsa_hagelloch_GLCTvsBoxcar.md): The 1861 measles outbreak in Hagelloch, Germany.
